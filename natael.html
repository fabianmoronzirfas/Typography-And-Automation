swissd.jsx

with swissd.jsx you can generate a width-fixed typo.

all you have to do, is to customize the config-object in the script and run it.

several customizations are possible: + add a shadow + add shear + add a rotation + improve the contrast of the biggest words + add a vignette (gradient with transparency) + add a perspective (3D)

https://github.com/natael/swissd
