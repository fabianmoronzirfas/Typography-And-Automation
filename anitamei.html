myRPG

This is the first start of a script for a simple computer role-playing game. The users task is it to find its way from home to the university campus. While traveling the user has to answer questions which are effecting its destiny. Going through the questions step by step the InDesign document builds up a text frame describing the user's journey. An info-graphic on the bottom of the document will show the user's journey.
After completing all quest the user can print the file as an A3 document.
